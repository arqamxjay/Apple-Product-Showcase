- **📱 iPhone Showcase** - iPhone 15 Pro Max, iPhone 15, iPhone 14, iPhone SE, iPhone 13

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Safari, Firefox, Edge)
- No server or build tools required

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/arqamxjay/Apple-Product-Showcase.git
   cd Apple-Product-Showcase
   ```

2. **Open in browser:**
   ```bash
   open index.html
   # or double-click index.html in your file explorer
   ```

3. **View the website:**
   - Local: `file:///path/to/Apple-Product-Showcase/index.html`
   - Live: Visit the GitHub Pages link above

## 🔧 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --apple-bg: #ffffff;
    --apple-text: #1d1d1f;
    --apple-blue: #0071e3;
}
```

### Update Product Images
Replace image URLs in `index.html`:
```html
<img src="your-image-url.jpg" alt="Product">
```

### Modify Content
Simply edit text in the HTML sections:
```html
<h1>Your Product Name</h1>
<p>Your product description</p>
```
